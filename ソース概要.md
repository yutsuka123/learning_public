# ソース概要.md

[重要] 本ドキュメントは「どこに何のソースがあるか」をフォルダ単位で説明します。
[厳守] フォルダを追加/大きく構成変更したら更新します。理由: 変更点の見落としを防ぐため。

## ルート直下
- `c/`: C の学習用サンプル
- `cpp/`: C++ の学習用サンプル（CMake/Visual Studio など）
- `cpp_m/`: モダンC++（C++11/14/17/20/23/modern）の学習用サンプル（単体コンパイル前提）
- `csharp/`: C# の学習用サンプル
- `java/`: Java の学習用サンプル
- `kotlin/`: Kotlin の学習用サンプル
- `python/`: Python の学習用サンプル
- `typescript/`: TypeScript の学習用サンプル
- `rust/`: Rust の学習用サンプル（Cargo）

## C#
- `csharp/modern-features/`
  - モダンC#（C# 9以降〜最新）の学習用サンプル
  - `ModernCSharpApp/`: ラムダ式（キャプチャ差）と最新機能の小例をConsole出力で確認

## C++ Modern
- `cpp_m/`
  - `main.cpp`: このフォルダの入口（コンパイラ/標準バージョンの確認と案内）
  - `cpp11.cpp`, `cpp14.cpp`, `cpp17.cpp`, `cpp20.cpp`, `cpp23.cpp`: バージョン別の学習サンプル（各ファイルを該当標準で単体ビルド）
  - `modern.cpp`: 最新（原則 C++23）を前提に、主要機能を総合的に扱うサンプル
  - `INSTALL.md`, `README.md`: ビルド/実行手順と学習の進め方

## Rust
- `rust/hello_world/`
  - Cargo プロジェクト（`Cargo.toml`, `src/main.rs`）
  - `cargo run -- [name]` で HelloWorld を表示
