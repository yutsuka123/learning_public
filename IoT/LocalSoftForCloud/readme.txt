IoT/LocalSoftForCloud readme

[重要]
- このフォルダは、ローカル環境とクラウド環境を同期する中継ソフトを格納する。
- 設定取得、指令情報取得、実行結果報告を双方向で扱う。
理由: Local運用とCloud運用の責務を分離し、段階移行を容易にするため。

[厳守]
- 同期方向は双方向を前提とする。
- 送受信IFは `IoT/IF仕様書.md` と同期して定義する。

[禁止]
- クラウド機密情報（鍵、トークン、シークレット）を平文保存しない。
理由: 認証情報漏えいを防止するため。

[変更履歴]
- 2026-03-07: 新規作成。双方向同期モジュールとして役割を定義。
