IoT/CloudServer readme

[重要]
- このフォルダは、将来クラウドに構築するサーバーのプログラム、設定、デプロイ関連ファイルを格納する。
- クラウド運用に必要なIaC、デプロイスクリプト、サーバー設定テンプレートを管理対象とする。
理由: クラウド移行時に実装と運用手順を同時に追跡できるようにするため。
