IoT/LocalSoft readme

[重要]
- このフォルダは、ESP32とMQTT経由で通信し、状態監視と命令送信を行うローカルソフトを格納する。
- デバイス状態の一覧表を保持し、必要に応じてサーバーとも通信する機能を持つ。
理由: 日常運用で使う監視・操作機能を独立して拡張しやすくするため。

[推奨]
- MQTTトピック・ペイロードは `IoT/MQTTコマンド仕様書.md` と同期して更新する。
- 一覧表示項目は `IoT/機能仕様書.md` に定義してから実装する。

[禁止]
- MQTT通信処理とUI処理を密結合にしない。
理由: 将来の保守性と試験容易性を維持するため。

[変更履歴]
- 2026-03-02: 新規作成。ローカル運用ソフト格納先として定義。
