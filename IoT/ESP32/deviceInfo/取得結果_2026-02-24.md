# ESP32-S3 取得結果（2026-02-24）

[重要] 本ファイルはCOM接続した実機から取得した情報の記録です。  
理由: 実装前の実機状態（チップ情報・フラッシュ情報・eFuse状態）を証跡として残すため。

## 1. 接続情報
- 取得日時: 2026-02-24
- 接続ポート: `COM9`

## 2. チップ基本情報
- Chip: `ESP32-S3 (QFN56) (revision v0.2)`
- Features: `WiFi, BLE, Embedded PSRAM 8MB (AP_3v3)`
- Crystal: `40MHz`

## 3. Flash情報
- Detected flash size: `16MB`
- Flash type set in eFuse: `quad (4 data lines)`
- Flash voltage set by eFuse: `3.3V`

## 4. eFuse（要点）
- Flash Encryption: `Disable`（`SPI_BOOT_CRYPT_CNT = 0b000`）
- Secure Boot: `False`（`SECURE_BOOT_EN = False`）
- Security Download Mode: `False`（`ENABLE_SECURITY_DOWNLOAD = False`）
- Download Mode Disable: `False`（`DIS_DOWNLOAD_MODE = False`）

## 5. 補足
- ROM容量そのものはSoC固定仕様であり、実行時の可変値としては通常取得しない。
- RAM空き量やヒープ情報はFW実行時API（`heap_caps_get_free_size` など）で取得する。
