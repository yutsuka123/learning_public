# ProductionTool `scripts`

[重要] 本フォルダは `ProductionTool` の開発補助スクリプト格納先です。  
理由: ビルド補助、試験補助、ログ採取補助を本体実装から分離するため。

## 想定対象
- 起動確認補助
- ログ採取補助
- テストデータ準備補助
- インストーラ作成補助

## 実装時の注意
- [厳守] セキュリティ解除を前提にした一時しのぎスクリプトを常設しない。
- [厳守] 実運用の秘密値を引数やファイルへ平文保存しない。
- [推奨] 実行手順は `IoT/コマンド仕様書.md` と同時更新する。
