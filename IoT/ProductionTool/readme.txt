ProductionTool folder

[重要] このフォルダは LocalServer と分離した工場内専用ソフトウェア ProductionTool の実装格納先です。
理由: 通常運用ソフトと製造用高リスクソフトを明確に分離するため。

[厳守] LocalServer と ProductionTool は別ソフト、別実行物、別インストーラとして扱います。
理由: 顧客向け通常運用と工場内専用操作の責務を混在させないため。

[厳守] Rust 共通化は LocalServer 側で整備した SecretCore の共通部や関連ライブラリを再利用する意味であり、ソフトウェア自体を統合する意味ではありません。
理由: 共通化と分離を両立し、手戻りを減らしつつ安全境界を保つため。

[推奨] 実装追加時は更新日時と変更理由を本ファイルへ追記してください。
