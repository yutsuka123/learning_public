# ProductionTool `src`

[重要] 本フォルダは `ProductionTool` の実装本体格納先です。  
理由: 起動処理、画面遷移、`SecretCore` 共通部接続、dry-run、安全停止をコード上で分離して管理するため。

## 想定責務
- 起動画面と状態表示
- 追加認証
- 対象機確認
- 実行内容確認
- dry-run 導線
- 監査ログ参照導線
- エラー / 安全停止

## 実装時の注意
- [厳守] `LocalServer` と同一プロセス化しない。
- [厳守] 不可逆処理本体は `004-0008` の段階で実装しない。
- [厳守] 秘密値を UI 表示用 DTO へ混ぜない。
- [推奨] 画面状態、認証状態、対象機状態、監査ログ状態を別モジュールに分ける。
