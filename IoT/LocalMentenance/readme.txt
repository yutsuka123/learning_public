IoT/LocalMentenance readme

[重要]
- このフォルダは、OTAや保守作業を行うGUIソフトを格納する。
- サーバーと通信し、保守操作（更新、診断、状態取得）を実行できる機能を持つ。
理由: 運用保守で使用する操作系を実装系から分離し、安全に管理するため。

[厳守]
- 保守操作の実行ログを取得できる設計を採用する。
理由: 問題発生時の追跡性を確保するため。

[新規利用禁止]
- 本フォルダへESP32ファームウェア本体のソースを配置しない。
理由: 保守GUIとデバイスファームの責務を明確に分離するため。

[変更履歴]
- 2026-03-02: 新規作成。OTA/保守GUI格納先として定義。
