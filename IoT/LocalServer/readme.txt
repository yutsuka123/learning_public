IoT/LocalServer readme

[重要]
- このフォルダは、ローカルサーバーとして動作するHTTPサーバー実装を格納する。
- ESP32、LocalSoft、LocalMentenance から利用されるローカルAPIを提供する。
理由: ローカル環境での制御・監視系機能を単一の入口に集約するため。

[推奨]
- API仕様は `IoT/IF仕様書.md` と同期して管理する。
- 起動方法・設定方法は今後このフォルダに手順ファイルを追加して管理する。

[禁止]
- 開発用の暫定設定を本番設定として固定化しない。
理由: 後戻り忘れによる運用障害を防ぐため。

[変更履歴]
- 2026-03-02: 新規作成。ローカルHTTPサーバー格納先として定義。
